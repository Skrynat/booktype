# booktype
A terminal-based typing application for reading books through active typing, inspired by Monkeytype. Track progress and accuracy while reading novels, technical books, and documentation.

## Goal

Turn reading into active typing:
- dim upcoming text
- correct typed text becomes bright
- mistakes become red
- backspace to fix errors
- track progress and accuracy
- WPM (optional)

## Future features

- TXT support
- EPUB support
- save progress
- chapter mode
- terminal UI
- Cyberdeck-friendly workflow
